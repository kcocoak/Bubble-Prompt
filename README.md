#Bubble Prompt (泡泡提示器)可生成询问ai提示词

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![React](https://img.shields.io/badge/React-19-61dafb.svg)
![Vite](https://img.shields.io/badge/Vite-6.0-646cff.svg)

**Bubble Prompt** 是一款专为中文用户打造的现代化提示词生成工具。它摒弃了枯燥的表单，采用灵动的“泡泡”设计语言，帮助你轻松构建高质量的 AI 提示词。

> **核心理念**：让灵感像泡泡一样自由升起。

## ✨ 核心特性

- **🫧 泡泡美学 (Bubble Aesthetic)**
    - 动态流光背景，营造沉浸式体验。
    - 全局 Glassmorphism（磨砂玻璃）设计。
    - 细腻的悬浮与回弹动画，交互 Q 弹有趣。

- **🧠 CO-STAR 专业内核**
    - 内置 **CO-STAR (Context, Objective, Style, Tone, Audience, Response)** 专业提示词框架。
    - **全中文优化**：从底层逻辑到界面全中文适配，拒绝“翻译腔”。
    - **智能分析**：自动识别任务类型（代码、文案、策划），推荐最合适的问题。

- **📂 本地灵感盒子**
    - 生成的优质 Prompt 可一键保存到本地。
    - 使用 LocalStorage 存储，保护隐私，无需联网。

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

打开浏览器访问 `http://localhost:5175` 即可使用。

## 🛠️ 技术栈

- **Frontend**: React + Vite
- **Styling**: Native CSS Variables (No UI Framework dependency)
- **Animation**: CSS Keyframes + Transitions
- **Font**: Google Fonts (Fredoka, Nunito)
